# 定義
DS102PythonSampleは、Pythonを使用してDS102/DS112コントローラを制御するための
参考として作成したサンプルプログラムです。
複雑な構成にはせず最低限必要な部分を抽出したシンプルな形で作成しています。
サンプルは自由に利用することができますが、
お客様のプログラムの内容や動作につきましては弊社では責任を負いませんのでご了承下さい。

# 概要
プログラムを起動後、「通信ポート」からDS102/DS112コントローラと接続するCOM番号を選択し、
「接続」ボタンを押して下さい。
「- (CCW)」「+ (CW)」ボタンを押すとDS102/DS112コントローラへ駆動コマンドを送信しステージを駆動します。
便宜上、駆動の度に駆動速度設定をしていますが、1度設定すれば毎回設定する必要はありません。
駆動中はDS102/DS112コントローラからステータスを受信し、「ステータス」に駆動状態を表示します。
